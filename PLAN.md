# Build Plan

## Goal

Build a working AI agent demo using Nemotron in roughly two hours.

## Timeline

### 0-15 min

- Finalize the core use case
- Assign roles

### 15-60 min

- Integrate the Nemotron API
- Lock the system prompt and output schema

### 60-90 min

- Build the frontend input and output UI
- Render structured response cards

### 90-120 min

- Prepare the demo flow
- Test 2-3 scenarios
- Polish copy and visuals

## Success Criteria

- Accepts a freeform user situation
- Returns:
  - risk level
  - actions
  - message
- Demo feels clear in under 30 seconds
